## 布局

```
┌ TopBar ─────────────────────────────────────────┐
├ LeftRail (~240px) ─┬─ MainStage (flex) ──────────┤
│ 任务历史            │  同一时刻仅一种视图：          │
│ 当前任务状态        │  Upload | Scanning |         │
│ + 新建任务          │  Conclusion | Done | Entities│
│ 我的实体 (底部)     │                              │
└────────────────────┴──────────────────────────────┘
```

## 主舞台状态机

| 条件 | 主区显示 |
|------|----------|
| 无 job | Upload 空态 |
| draft | Upload + 文件名 +「开始扫描」 |
| scanning/queued | 进度 |
| scanned + 结论页打开 | **ConclusionView** 全屏 |
| scanned + 结论关闭 | Upload 区 + **banner**：「N 个实体待确认 → 查看结论」 |
| done | **CompletionView** |
| 左栏点「我的实体」 | **MyEntities** 占主区 |

## 结论页（选项 C）

- Worker 在线扫完 → **自动进入** ConclusionView
- 用户点「关闭/返回」→ 回到 scanned + banner，**不丢实体**
- 刷新恢复 scanned → **仅 banner**，不自动再弹结论页
- Worker 离线：有已记住实体 → 自动脱敏；0 实体 → ConclusionView 手动添加

## 历史任务

- 点击 done → **主区 CompletionView**（非抽屉）
- 左栏高亮该任务

## 视觉令牌 v2.1（2026-06-01 调色升级）

| Token | 值 | 用途 |
|-------|-----|------|
| `--deid-bg` | `#f7f6f3` | 页面底（Notion 暖灰） |
| `--deid-surface` | `#ffffff` | 卡片/面板 |
| `--deid-rail-bg` | `#fafaf9` | 左栏背景 |
| `--deid-ink` | `#37352f` | 主文字 |
| `--deid-ink-secondary` | `#6f6e69` | 次要文字 |
| `--deid-primary` | `#5e6ad2` | **主按钮/进度条**（Linear indigo，非纯黑） |
| `--deid-border` | `#e9e9e7` | 分隔线 |
| `--deid-primary-soft` | `rgba(94,106,210,0.1)` | 选中/hover 背景 |
| `--deid-shadow-sm` | `0 1px 2px rgba(15,15,15,0.04)` | 面板微阴影 |

**原则**：主操作一律 indigo；纯黑仅用于 ink 文字；badge 语义色走 token；上传区用 SVG 图标不用 emoji。
